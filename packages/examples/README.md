# Examples

This directory contains example implementations demonstrating how to use the Multi-Agent Governance Framework.

## Available Examples

### basic-usage.ts
Demonstrates basic framework usage, including:
- Importing core types and utilities
- Working with agent statuses
- Understanding message types

## Running Examples

```bash
# Install dependencies first
npm install

# Run an example with ts-node or tsx
npx tsx packages/examples/src/basic-usage.ts
```
